# Chaika Tickets — Pretix theme

Кастомне оформлення для вашого Pretix-магазину
(`https://chaikatickets.org/test-chaika/`). Тема не змінює функціонал —
лише дизайн. Календар, події і всі сторінки залишаються рідними Pretix.

## Як підключити (5 хвилин)

1. Зайдіть в Pretix-адмінку як власник.
2. **Organizer-рівень** (стилізує одразу всі івенти):
   `Settings` → `Customize` → знайдіть поле **«Custom CSS»**.
3. Або **Event-рівень** (тільки для одного івенту):
   `<Event>` → `Settings` → `Customize` → **«Custom CSS»**.
4. Скопіюйте увесь вміст файлу [`chaika.css`](./chaika.css) і вставте
   у це поле. Натисніть **Save**.
5. Оновіть публічну сторінку — дизайн застосується миттєво.

> Якщо у вашій версії Pretix поля «Custom CSS» немає, увімкніть
> плагін **«Customizable styling»** (`pretix_customcss`) у
> `Plugins` → активуйте → перезавантажте сторінку Settings.

## Налаштування мов (EN за замовчуванням, RU, UK)

Pretix сам показує перемикач мов, якщо їх увімкнено в івенті.

1. Перейдіть у потрібний івент → `Settings` → `General`.
2. Знайдіть секцію **«Languages»** (Мови / Sprachen).
3. Поставте галочки на:
   - **English** ← оберіть як **Default language**
   - **Russian (русский)**
   - **Ukrainian (українська)**
4. Збережіть.

Після цього у шапці магазину з’явиться перемикач `EN / RU / UK`.
Тема `chaika.css` уже стилізує його під фірмові пілюлі.

> Те саме можна зробити на рівні **Organizer** —
> `Organizer Settings` → `Locales` — щоб мовні налаштування
> успадковувались усіма івентами.

## Що дає тема

- Фірмова палітра Chaika (блакитно-білий градієнт `#4FB3F0 → #1E7FBF`)
- Легкий «3D» через мʼякі тіні, нахили на hover і шар сітки на тлі
- Сучасні pill-таби `Список / Тиждень / Місяць / День / iCal`
- Картки подій із градієнтним акцентом і силуетом чайки
- Календар із плитковим виглядом, підсвітка «сьогодні»
- Кругла кнопка «Купити квиток» з glow-ефектом
- Стилізований перемикач мов EN/RU/UK
- Мобільна версія + підтримка `prefers-reduced-motion`

## Брендинг

Усі кольори зібрані у `:root` зверху файлу — змінюйте одне значення
і вся тема перебудується. Логотип — інлайновий SVG у CSS, не залежить
від хостингу зображень.

## Видалити тему

Просто очистіть поле «Custom CSS» і збережіть — Pretix повернеться
до дефолтного оформлення.
